# Introduction

## Installing Teh's Core Mod

Download the latest release from the [Nexus page](https://www.nexusmods.com/stardewvalley/mods/3256) or install it via ModDrop. Alternatively, you may clone the repository and compile it yourself. You will need Visual Studio 2019 or the latest version of MSBuild. Mono may be unable to compile the mod due to the use of newer language elements.
